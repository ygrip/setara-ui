# setara-ui

SvelteKit frontend for Setara.

## Repository

```text
git@github.com:ygrip/setara-ui.git
```

## Local Development

```sh
npm install
npm run dev
```

## Environment

```sh
VITE_SETARA_API_BASE_URL=http://localhost:8080
VITE_SETARA_WS_BASE_URL=ws://localhost:8080
```
